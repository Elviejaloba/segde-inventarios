Prompt para Replit (React + Firebase + Tailwind)
Crea una aplicación en React con Tailwind CSS y Firebase que permita gestionar un checklist de seguimiento de pedidos de muestreos para la temporada de invierno.

Requisitos principales:
Interfaz responsive y elegante.
Animaciones:
Indicar progreso cuando un checklist esté completo o vaya por la mitad.
Selección de sucursal:
Antes de acceder, el usuario debe seleccionar su sucursal de una lista desplegable.
Sistema multiusuario:
Varios usuarios pueden editar el checklist sin afectar otras sucursales.
Ranking de cumplimiento:
Las sucursales deben ordenarse por cantidad de artículos enviados y comunicados.
Checklists interactivos:
Cada sucursal tiene su propio checklist con códigos solicitados.
Guardado en tiempo real:
Firestore para almacenar cambios sin interferencias entre usuarios.
Filtros y búsqueda:
Opción de buscar códigos específicos dentro del checklist.
Dashboard de visualización:
Cuadro de seguimiento con progreso en tiempo real.
Edición abierta:
Cualquier usuario con acceso puede modificar el checklist de su sucursal sin afectar a otras sucursales.
Botón de comunicación vía WhatsApp:
Agregar un botón al final de la aplicación que permita compartir el seguimiento del pedido vía WhatsApp.
El botón debe generar un mensaje con la información del checklist y redirigir a WhatsApp.
Incluir la siguiente leyenda:
"Esta es una herramienta de seguimiento. La comunicación debe realizarse vía mail adjuntando el comprobante de toma de inventario del artículo solicitado."
